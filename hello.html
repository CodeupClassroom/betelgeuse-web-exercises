Welcome to Codeup!

We're going to work with a variety of languages, methods, and industry tools

  • HTML - language of content and structure
  • CSS - language for styling and layout
  • JS - language for client side interactivity
  • MySQL - database language for storing data for later
  • Java - language for server side logic
  • Spring - framework for Java that saves you time
  • Paired Programming: where two devs work together to solve the same problems.
  • Version control with git

Client side or Front End

Client side means your browser (or mobile device)

This is the part of an application people see and interact with

Presents an interface to people

Server side or Back End

Servers serve requests

Performs any business critical logic
